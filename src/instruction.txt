Версия Java-21

Инструкция запуска программы
1. Откройте терминал(cmd)
2. Перейдите в директорию, в которой находится файл util.jar и текстовые файлы in1.txt, in2.txt (Названия файлов используются для примера)
3. Пропишите команду в консоли
java -jar util.jar
Опции, которые вы можете использовать:
-p - позволяет применить префикс к выходным файлам
-o позволяет сохранить выходные файлы в заданную директорию
-f - выводит полную статистику по данным в файлах
-s - выводит краткую статистику по данным в файлах
-a - позволяет добавлять результат в конец существующих файлов
